## To contribute to this repository you should 
1. Make an icon in windows aero style, or similar to it;
2. Make a pull request;
4. Put your icon in the "icons" folder and add all the necessary support in [icons.json](https://github.com/o-dka/vsclassic-icon-theme/blob/main/icons.json) file;
5. Wait for our response! We will usually try to respond in the next 24hrs.
